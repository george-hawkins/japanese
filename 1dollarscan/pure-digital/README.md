Chapter start sample
====================

I reconstructed the `crop.png` used in my experiments as a pure digital document so one could compare a perfect 600 DPI image of the text with the results of the scan done at 600 DPI.

I used Apple's [Pages work processor](https://en.wikipedia.org/wiki/Pages_(word_processor)) to produce the text. I used a far larger font size (hence the references below to 328 DPI and 321 DPI rather than 600 DPI). In the original the chapter header is a different font size to the chapter text, hence the slightly different DPIs used below to replicate this.

Files:

* [`chapter-text.pages`](chapter-text.pages) - chapter text in Pages format.
* [`chapter-text.pdf`](chapter-text.pdf) - ditto as PDF (and exported PNG at 328 DPI for use in `.xcf`).
* [`chapter-title.pages`](chapter-title.pages) - chapter title in Pages format.
* [`chapter-title.pdf`](chapter-title.pdf) - ditto as PDF (and exported PNG at 321 DPI for use in `.xcf`)).
* [`chapter-title-good-furigana.pdf`](chapter-title-good-furigana.pdf) - chapter title without the encircled digit pushing the furigana right.
* [`chapter-title-good-furigana.png`](chapter-title-good-furigana.png) - ditto as PDF (and exported PNG at 321 DPI for use in `.xcf`)).
* [`chapter-start.xcf`](chapter-start.xcf) - PNGs produced from PDFs combined into a single image.
